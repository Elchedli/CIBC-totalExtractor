# How this functions : 
1 - take the code in @extractFromChrome when you open a credit card and put the 12 months period.  
2- let the program finish and it will a json file & put them in data folder with `credit_card.json` & `debit_card.json`.  
3- redo 1 & 2 for the other credit and debit cards.  
4- use calculateCardSum.js to calculate get the total of each and add the sums manually.  
5- Congrats, you got the final a manual calculation to verifiy if the sum that the bank is showing is correct or to calculate the pendings with it without mistake.
